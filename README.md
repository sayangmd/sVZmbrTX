# 项目简介

## 核心功能
- 基于YOLOv8的堵车检测系统，支持图片、视频及摄像头进行检测
- 提供完整源码、1900张标注数据集、训练好的模型及训练结果
- 支持选择检测模型，自定义置信度和IOU阈值
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存功能
- 提供完整项目运行教程

## 适用场景
- 交通流量监控与分析
- 城市交通拥堵实时检测
- 智能交通管理系统开发
- 道路状况监测与研究
- 视频分析中的交通事件识别

## 技术特点
- 采用Python 3.9开发，集成PyQt5用户界面
- 环境依赖包括opencv-python、ultralytics、torch1.9等
- 提供Pycharm+Anaconda或VSCode+Anaconda开发环境
- 支持UI界面操作，包含图标资源文件
- 模型训练完成，配置环境后可直接使用
